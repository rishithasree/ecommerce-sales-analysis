# E-Commerce Sales Data Analysis

## Project Overview

This project analyzes e-commerce sales data to identify sales trends, top-selling products, and city-wise revenue patterns. The analysis helps understand customer purchasing behavior and business performance using data visualization and exploratory data analysis techniques.

## Tools and Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab / Jupyter Notebook

## Project Structure

```
ecommerce-sales-analysis
│
├── dataset
│   └── sales_data.csv
│
├── notebook
│   └── analysis.ipynb
│
├── images
│   └── charts.png
│
└── README.md
```

## Analysis Performed

* Data Cleaning and Preprocessing
* Monthly Sales Trend Analysis
* Top-Selling Products Identification
* City-wise Revenue Analysis
* Data Visualization using charts and heatmaps

## Key Insights

* Identified months with the highest sales revenue
* Determined the most popular products based on quantity sold
* Compared revenue across different cities

## Conclusion

This project demonstrates practical data analysis skills including data cleaning, exploratory data analysis, and visualization. It shows how raw sales data can be transformed into meaningful insights for business decision-making.
